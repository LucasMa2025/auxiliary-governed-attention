## 一、概述

本文档定义 AGA 系统的完整可视化方案，涵盖：

1. **ANN 检索可视化** - 向量空间、候选分布、检索路径
2. **路由决策可视化** - Gate-0/1/2 决策流程、Top-k 选择
3. **知识状态可视化** - 注入、确认、隔离、移除等生命周期变化
4. **实时监控仪表盘** - 系统健康、性能指标、告警

## 八、数据模型

### 8.1 向量可视化数据

```typescript
interface VectorPoint {
  lu_id: string;
  x: number;  // 投影后 X 坐标
  y: number;  // 投影后 Y 坐标
  distance?: number;  // 到查询点的距离
  lifecycle_state: string;
  condition_preview: string;
}

interface VectorSpaceData {
  query_point?: VectorPoint;
  candidates: VectorPoint[];
  all_vectors?: VectorPoint[];  // 全量向量 (可选)
  projection_method: 'tsne' | 'umap' | 'pca';
  statistics: {
    total_vectors: number;
    visible_vectors: number;
    projection_time_ms: number;
  };
}
```

### 8.2 路由追踪数据

```typescript
interface RouteTrace {
  trace_id: string;
  timestamp: string;
  gate0: {
    decision: 'PASS' | 'BYPASS' | 'DISABLED';
    namespace: string;
    reason: string;
  };
  gate1: {
    decision: 'PASS' | 'BYPASS';
    confidence: number;
    threshold: number;
    entropy: number;
  };
  gate2: {
    selected_slots: Array<{
      slot_idx: number;
      score: number;
      lu_id: string;
    }>;
    total_candidates: number;
    top_k: number;
  };
  final: {
    aga_applied: boolean;
    fusion_weight: number;
    latency_ms: number;
  };
}
```
